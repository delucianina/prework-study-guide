HTML

  • I learned HTML5 at Drexel, but never used it for work after that.
  • Codecademy has an excellent HTML course and I took it in April as review.
  • The head element contains info about the webpage, like the title that appears in the tab.
  • The body element contains the content that will display on the webpage.

CSS

  • I also learned CSS at Drexel, but never used it for work after that.
  • Codecademy has a great CSS course too and I took it in May as review.
  • CSS allows us to style a webpage and make it responsive to different screensizes.

Git and GitHub

  • GitHub's desktop app is nice for beginners. I used it for game development at my last job.
  • Using it through the terminal is new to me.
  • Git helps maintain organization across a team of developers.

JavaScript

  • I have played around with JavaScript using Codecademy. I left off while learning about Objects.
  • Object oriented programming is important in game development.
  • JavaScript can be used to add interactivity to websites.
